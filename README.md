# ai-lit UI library (monorepo)

## Requirements

- Node.js 20+
- pnpm (see `packageManager` in root `package.json`)

## Commands

```bash
pnpm install
pnpm -r build
pnpm -r dev
```

## Packages

- `packages/ui`: Lit + TypeScript component library, builds ESM/CJS/UMD/IIFE.
- `apps/playground`: Vite app to validate components locally.
